# visit_card_template
## Установка и запуск
**Требуется Node.js и npm!!!**
- Клонирование репозитория:
```bash
git clone https://github.com/muz-muzzy/visit_card_template.git
```
- Переход в директорию проекта:
```bash
cd visit_card_template
```
- Установка зависимостей:
```bash
npm install
```
- Запуск
```bash
npm start
```
